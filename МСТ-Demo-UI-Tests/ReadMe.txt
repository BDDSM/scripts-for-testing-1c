В этой папке хранятся готовые сценарные тесты (папка Scripts), библиотека скриптов (папка Lib) для инструмента "Менеджер сценарного тестирования".
Проект находится по адресу: https://github.com/ivanov660/TestingTool-3.
Описание:
Scripts - папка, в которой хранятся готовые сценарии.
Scripts\Демо - хранятся скрипты, которые должны выполниться практически на любой базе 1С, где есть следующие справочники и документы: контрагенты, договоры контрагентов, реализации товаров и услуг.
Scripts\УТ11 - хранятся скрипты, для выполнения на базе УТ11. Могут выполняться и на других совместимых конфигурациях ЕРП, КА.
Lib - папка, в которой хранятся библиотека наборов действий выполняющих какую-либо операцию, к примеру, создание заказа клиента или расходного ордера на товары.
