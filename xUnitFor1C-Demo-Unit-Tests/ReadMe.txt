Размещены демо тесты, которые работают с Фреймворком xUnitFor1C. 
Описание:
Тесты_Unit_Шаблон.epf - шаблон обработки для серверного теста
Начальная загрузка данных - папка с обработками, которые загружают начальные данные.
Демо - папка с универсальными демо примерами, которые доллжны работать практически на любой базею
УТ11 - папка с тестами, которые будут работать в конфигурациях УТ11 или ЕРП2.
